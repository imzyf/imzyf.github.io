解决 Ubuntu Sogou 无法选词输入中文

sogou 输入法突然无法选词输入中文，候选词位置出现白框，多次重重装 fcitx 和 sogou 也没有解决。尝试使用 google pinyin 代替，但是感觉很不顺手。

issue in GitHub

sogou 输入法 GitHub 上的一些 issue：

  • #43
  • #177
  • #179

解决

方案一

try to lastest version

方案二
