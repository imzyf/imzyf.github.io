Ubuntu 命令行下免密码执行 sudo 命令

April 10, 2017 · 1 min · 27 words · Me
Table of Contents
  • References

解决你的问题的方法是将你的用户加入 sudoers 文件。

sudo visudo

在文件底部输入：

username ALL=(ALL) NOPASSWD: ALL

这只适用于终端窗口中的 sudo 命令。例如，当你尝试在 software center 中安装软件包时，将提示你输入密码。
