配置

其中 GitHub 上有的就不再累述了，主要注意：

  • 添加 R.generated.swift 不要勾选 Copy items if needed，软引用就好。
  • 要在 .gitignore 添加 *.generated.swift 以避免不必要的冲突。

添加新的资源文件后需要 command + b 编译下，才可以使用。
