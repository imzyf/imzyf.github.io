在 CentOS 编译安装 PHP

April 29, 2020 · 1 min · 60 words · Me
Table of Contents
  • 环境
  • 下载 PHP
  • 安装相关库
  • 配置
  • 编译安装

2023-12-22：内容不再维护，推荐使用 Docker。

环境#

  • CentOS 7.4
  • PHP 5.6.40

下载 PHP#

源码地址：https://www.php.net/releases/#5.6.40

cd /tmp
wget https://www.php.net/distributions/php-5.6.40.tar.gz --no-check-certificate
tar zxvf php-5.6.40.tar.gz

安装相关库#

yum groupinstall "Development tools"

配置#

核心配置选项列表 | php.net

# 查看配置参数
./configure --help
./configure --help | grep openssl

# 注意替换 --prefix
./configure --prefix=/usr/local/php56 \
--with-openssl \
--enable-mbstring \
--enable-ftp

编译安装#

# 4核编译
make clean && make -j4
make install
