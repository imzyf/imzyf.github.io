Yii2 源码阅读 09 - Behavior

Behavior

参见：行为（Behavior）

Behavior 行为，也称为 mixins。 可以无须改变类继承关系即可增强一个已有的 组件 类功能。 当行为附加到组件后，它将“注入”它的方法和属性到组件， 然后可以像访问组件内定义的方法和属性一样访问它们。

yii behaviors

namespace yii\behaviors;

base ActionFilter

参见：过滤器（Filters）

过滤器本质上是一类特殊的 行为，所以使用 filter 和使用 behavior 一样。

namespace yii\filters;

RateLimiter 详解

RateLimiter 实现了一个基于 漏斗算法 的速率限制算法。

当 user 超过他的速率限制时 RateLimiter 将会抛出 TooManyRequestsHttpException 异常。

RateLimiter 需要 user 来实现 RateLimitInterface。 如果 user 未设置或未实现 RateLimitInterface，RateLimiter 将不执行任何操作。
