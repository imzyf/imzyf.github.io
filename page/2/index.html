插入排序 Insertion Sort

2018-11-24 阅读次数 0

将一个数组从高到低或者从低到高排序。

插入排序算法的工作原理：

  1. 将若干数字放在一个数组里，数组是乱序的。
  2. 从数组中挑选一个数字，它是哪个并不重要，但是为了方便我们挑选数组头部的这个。
  3. 将这个数字插入到一个新的数组里。
  4. 从乱序数组里挑选下一个数字也将它放到新数组里。这个数字要么在第一个数字前或者后，所以这个两个数字是被排序的。
  5. 再次重从乱序数组里挑选下一个数字也将它放到新数组里，并将数字放在正确的位置。
  6. 一直如此进行直到乱序数组中没有数字。这时也将等到一个排序好的新数组。
