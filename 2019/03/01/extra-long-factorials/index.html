打印 n! 的结果（1 <= n <= 100）。注意：当 n > 20 时 64 位的 Int 将无法直接存储结果。

思路

  • 将大数字用 数组 形式表示。比如 987 使用 [9,8,7] 代替。
  • 每一位乘以 n，再进行进位操作，得到新数组。
