编辑 settings.json#

command + , 右上角点击 打开设置。

{
  // 通过整体放大窗口比例
  "window.zoomLevel": 1.1,
  // 字体
  "editor.fontSize": 14,
  // 软换行
  "editor.wordWrap": "on",
}

显示空格和 tab 符号#

Editor: Render Whitespace
控制编辑器在空白字符上显示符号的方式。
all

Settings 同步#

只需从左下角的设置齿轮启用内置同步。
